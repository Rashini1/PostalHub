# PostalHub - Sri Lanka Post Office Frontend (React + Vite + Tailwind CSS)

PostalHub is a modern, responsive frontend UI for a Post Office system built using **React**, **Vite**, and **Tailwind CSS**.  

 

## Features

* Modern, responsive UI
* Component based structure  
* Clean design using Tailwind CSS
* Fast development with Vite + React
* Easy to customize for any postal-related system

 

##  Tech Stack

| Technology  | Purpose                  |
| ----------- | ------------------------ |
| React       | Frontend UI framework    |
| Vite        | Fast development bundler |
| TailwindCSS | Styling & responsiveness |

 

 

##  Installation & Setup

1. Clone the project or download:

```
git clone https://github.com/yourusername/postalhub.git
```

2. Install dependencies:

```
npm install
```

3. Start development server:

```
npm run dev
```

4. Open browser and go to:

```
http://localhost:5173/
```


- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh

## React Compiler
